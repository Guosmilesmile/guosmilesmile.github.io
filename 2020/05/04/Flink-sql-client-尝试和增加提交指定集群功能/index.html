Flink sql client 尝试和增加提交指定集群功能

发表于 2020-05-04 | 分类于 Flink | 阅读次数：

笔者想要做一个实时计算平台，通过sql来减少开发，如果直接用jar包开发，是可以做精细化管理，但是开发速度就会受到限制。

目前有两种方式，一种是做成一个jar包，sql放入配置文件中，但是这样就没办法很好用上社区的功能，限制太大。另一个是使用flink-sql-client。
